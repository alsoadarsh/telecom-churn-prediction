## Analysis

### churn_analysis.Rmd
Full pipeline — data cleaning, EDA, five model builds,
threshold tuning, ROC/AUC comparison, retention strategy.
Open in RStudio and click Knit to reproduce.

**Before knitting:** place `Model_Building_Data.rda` and `Implementation_Data.rda`
in this folder. See [data/README.md](../data/README.md) for how to generate them.

### churn_analysis.html
Pre-rendered output. Open in any browser —
no R installation needed.
